# Grading Rubric

Total: 100

| Component | What is evaluated | Points |
| --- | --- | ---: |
| Strategy design | Coherence of cluster selection, synchronization structure, and system-level design | 25 |
| Pseudocode | Correctness, completeness, and clarity of system-level pseudocode | 20 |
| Solution reasoning and analysis | Quality of tradeoff reasoning, explanation of results, and discussion of limitations | 15 |
| Code implementation | Whether the submitted strategy runs correctly with the provided Ray simulator | 20 |
| Experimental results | Completeness and correctness of baseline comparisons and output files | 20 |

Grouping:

- strategy design + pseudocode + reasoning = 60
- code + experimental results = 40

Strong submissions usually show a clear tradeoff argument rather than only chasing the smallest training time.
